Project Toon Shading

3D Graphics

Project Overview

The Toon Shading project was an assignment for my CS250 Computer Graphics course at DigiPen. The objective was to implement various toon shading techniques in shaders and integrate them into an interactive graphics demo.

Tasks Completed

  • Developed a fragment file to render custom ranges toon.
  • Developed a fragment file to render uniform discretization(chunks) toon.
  • Developed a fragment file to render texture-based discretization(gradient texture) toon.
  • Added Anti-Aliasing in each type of toon shading.

Reflection

I struggled for a long time because the object’s color didn’t display as intended, and the shading was not properly rendered. However, I collaborated with my classmates who were working on the same assignment, exchanging ideas and discussing the problem together. Through these efforts, along with researching online, I was able to identify the cause of the issue. I checked the order of the formulas and the position of the light source, then corrected the formulas to solve the problem.
