Project Procedural Geometric Modeling

3D Graphics

Project Overview

The Procedural Geometric Modeling project was an assignment for my CS250 Computer Graphics course at DigiPen. The objective was to develop procedural mesh generation algorithms and integrate them into an interactive graphics demo.

Tasks Completed

  • Designed and implemented a graphics class.
  • Developed functions that creates a surface with stack and slices and draws lines from each point.
  • Developed functions to render three-dimensional shapes such as spheres and cones.

Reflection

When creating functions to render the cylinder and cone, I faced difficulties because the lines were not drawn correctly on the flat surface. Since I believed the logic of the function was incorrect, I rewrote the function with a new logic, and the problem was resolved. Also, there was an issue about emsdk that is complier for web. I had a misconfigured path for emsdk, but I resolved the issue by correcting the path.
